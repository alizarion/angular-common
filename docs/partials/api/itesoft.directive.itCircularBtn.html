itCircularBtn
directive in module itesoft since v1.1

Description

Provide a circular button like for the full screen button

  <it-circular-btn></it-circular-btn>

Usage

as element:
<it-circular-btn>
</it-circular-btn>
as attribute
<ANY it-circular-btn>
   ...
</ANY>
as class
<ANY class="it-circular-btn">
   ...
</ANY>
