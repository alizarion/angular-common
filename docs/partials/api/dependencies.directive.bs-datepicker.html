bs-datepicker
directive in module dependencies since v1.0

Description

AngularStrap Add datepicker functionality with any form text input. Datepickers require the tooltip module and dateParser helper module to be loaded. These docs currently use bootstrap-additions for styling purposes. See more : mgcrea-datepickers

Usage

as attribute
<ANY bs-datepicker>
   ...
</ANY>
