itSideMenus
directive in module itesoft since v1.0

Description

A container element for side menu(s) and the main content. Allows the left and/or right side menu to be toggled by dragging the main content area side to side.

To use side menus, add an <it-side-menus> parent element. This will encompass all pages that have a side menu, and have at least 2 child elements: 1 <it-side-menu-content> for the center content, and <it-side-menu> directives

it-width="400" attribute on it-side-menu directive Width of the side menu, in pixels, max value 800.
it-animate="true" on it-side-menu-header to animate de side menu open and close button
it-hide-button-menu="true" on it-side-menu-header to hide header menu Close/open button
it-side-menu-header headers are optionnal 
<it-side-menus>

 <it-side-menu-header it-animate="true"  it-hide-button-menu="true">
 </it-side-menu-header>

  <!-- Center content -->

  <it-side-menu-content>
  </it-side-menu-content>

  <!-- menu -->


  <it-side-menu >
  </it-side-menu>

</it-side-menus>

Usage

as element:
<it-side-menus>
</it-side-menus>
as attribute
<ANY it-side-menus>
   ...
</ANY>
as class
<ANY class="it-side-menus">
   ...
</ANY>
