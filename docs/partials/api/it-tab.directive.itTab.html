itTab
directive in module it-tab since v1.2

Description

Button that represent tab header

Attribute

Param Description
label Button Label
id tab identifier (must be the same that tab content id)

Exemple

   <it-tab
     label="'FEATURE.COMPANY.TITLE'"
     id="'validate-header-tab-company'">
   </it-tab>

Usage

as element:
<it-tab>
</it-tab>
