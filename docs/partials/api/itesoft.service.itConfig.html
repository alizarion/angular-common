itConfig
service in module itesoft since v1.2

Description

This is a configuration provider. It load a json file and let you inject everywhere, you can also override each properties of the configuration through url parameters.

Property Default value Description
configFile app/config/config.json Allows to override the default url to get JSON configuration object
defaultNamespace '' Set the default namespace of the configuration file when you use get() function
allowOverride false Allows to use URL parameters to override JSON configuration peroperties in URL ?key=value&key=value

Dependencies

  • $location
